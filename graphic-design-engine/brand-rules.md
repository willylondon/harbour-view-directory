# Studio Standards

1. Always read the client's brand kit first.
2. Never use generic stock-photo aesthetics — prompts should reference specific visual styles, moods, and compositional rules.
3. Print assets must always include bleed and be exported at 300 DPI.
4. Social assets must be exported as sRGB, not CMYK.
5. Logo files are never to be rasterized or resized — always use the original vector source.
