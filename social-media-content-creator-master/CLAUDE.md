## Platform Intelligence (2025–2026)

### TikTok
- **What it rewards:** Watch time (especially first 3s & completion), saves, and shares (over likes). Niche authority and consistent posting compound. Videos 60-90 seconds perform best if retention holds. Uses SEO on captions/voiceovers.
- **What kills reach:** Watermarks, off-niche posting, low completion rates, shallow engagement (likes but no saves/shares).
- **Hook Rule:** 0–2 seconds. Visual AND verbal simultaneously. Text overlay + spoken hook at the same time is king.

### Instagram
- **What it rewards:** DM Sends (Sends Per Reach) are the strongest signal. Watch time (passing the 3s mark) and rewatch behavior. Engagement rate (likes per reach). Carousels (up to 20 slides). Stories interactivity (polls/questions). Original content and keyword-rich captions.
- **What kills reach:** Reposting (Aggregator Penalty), editing after posting, watermarks.
- **Hook Rule:** First frame must stand alone. No landscape intros, logos, or title cards on frame one.

### YouTube
- **What it rewards:** Maximize viewer satisfaction (surveys, sentiment, completion). High CTR (5-10%) and Audience Retention (50%+ midpoint, 70%+ priority). Session continuation (watching another video). Replying to early comments (first 2 hours). S.T.A.R.T intro.
- **What kills reach:** Low retention (under 40%), clickbait (high CTR, low retention), inconsistent uploads, ignoring the Community tab.
- **Hook Rule:** Reveal the value in the first 10 seconds. State what the viewer will know/feel, then deliver it.

### X (Twitter)
- **What it rewards:** Conversation depth. Replies that the author replies to (+75 score). First-hour engagement velocity. Rich media (video/images).
- **What kills reach:** External links in the main post. Repetitive content. All-caps text. Ghosting the replies.
- **Hook Rule:** First line is everything. Leave the explanation for the body.

## The Hook Masterclass
1. **Curiosity Gap:** "I ordered from 8 ramen spots in Tokyo. Only 2 were worth it."
2. **Problem/Validation:** "Every tech reviewer does this wrong..."
3. **Bold Statement/Pattern Interrupt:** "Unpopular opinion: flagship phones aren't worth reviewing anymore."
4. **Outcome/Aspirational:** "How I got into every sold-out restaurant in Tokyo without a reservation."

## Script Architecture
- **Short Form (15-90s):** 0-2s Visual/Spoken Hook -> 2-15s Context Bridge -> 15-60s Core Value (fast cuts) -> Final 5s CTA.
- **Long Form (8-20m):** 0-30s Cold Open -> 30s-2m Context Hook (open loops) -> 2-10m Core Content (chapters) -> 10-15m Depth Layer -> Close/CTA.
- **X Thread:** Tweet 1 Hook -> Tweet 2-3 Evidence -> Tweet 4-8 Value -> Final Tweet CTA.

## Transcript & Video Analysis Workflow
Run these checks before writing new content:
1. **Hook Audit:** Analyze the first 3 seconds (verbal/visual hook).
2. **Retention Risk Map:** Flag drop zones (filler phrasing, dead air).
3. **Pacing Diagnosis:** Count beats per minute (cuts, new ideas).
4. **Hook-Payoff Alignment:** Does the end actually deliver on the start?
5. **CTA Effectiveness:** Make sure it is single-focused and specific.
6. **Platform Fit:** Tailor specifically for the destination platform.
